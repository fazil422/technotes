# Panes
```
o/e     Ctrl Shift    split pane horiz / vert
Tab     Ctrl (Shift)  next (prev) pane
Arrow   Alt           move to pane
RMB     Ctrl          layout drag (then release Ctrl to position)
```

# Tabs
```
t       Ctrl Shift    new tab
PgUpDn  Ctrl          switch tab
PgUpDn  Ctrl Shift    reposition tab
s       Ctrl Shift    scrollbar
w       Ctrl Shift    close pane
Arrow   Ctrl          resize
zx      Ctrl Shift    zoom / maximise
r       Cmd (Shift)   rotate
```

# Misc
```
gt      Cmd  (Shift)  broadcast group (ungroup) panes (all / tab)
i       Cmd           new win (new  pid)
i       Ctrl Shift    new win (same pid)
cv      Ctrl Shift    copy / paste
+-0     Ctrl          font size
f       Ctrl Shift    search
rg      Ctrl Shift    reset / reset and clear
q       Ctrl Shift    quit
F11                   fullscreen
```
